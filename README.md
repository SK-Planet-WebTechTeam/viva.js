# viva.js
a simple 2D Javascript Physics engine

Features
- world
- behavior
  - constant acceleration
  - boundary collision
  - body collision
- body
  - circle
  - rectangle (partial support)
- vector
 

Roadmap
- add more shapes
  - full support for rectanglular bodies
  - polygon
- add more behaviors
- improve accuracy
- body collecion
- add more renderers
  - DOM renderer
  - WebGL support
